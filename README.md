We can use [dotnet new](https://docs.microsoft.com/en-us/dotnet/core/tools/dotnet-new) to create a new project.
```
dotnet new --list
```

for example
```
cd console_example
dotnet new console
dotnet add package Newtonsoft.Json --version 13.0.1
dotnet run
```
